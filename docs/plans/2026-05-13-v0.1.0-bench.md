# 1bit.cashier v0.1.0-bench Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Ship `v0.1.0-bench` — a fresh-clone-runnable bench MVP that drives a scripted demo transaction through six Rust crates against fully mocked hardware, with a vanilla HTML/JS customer display rendering the transaction in real time over a websocket. Tagged as the first public release on GitHub.

**Architecture:** Six small Rust crates organized as a Cargo workspace. `cashier-bus` defines an `EventBus` trait (default impl: `tokio::sync::broadcast`) and the typed event envelope every other crate publishes. `cashier-order` is the spine — a fully unit-tested order state machine driven by events. `cashier-vision` / `cashier-audio` / `cashier-recycler` / `cashier-audit` each define a thin trait plus a mock implementation. A top-level `cashier-pos` binary wires everything together, runs a scripted demo when invoked with `--bench`, and exposes an `axum` websocket the customer-display PWA subscribes to.

**Tech Stack:** Rust 2024 edition · `tokio` (async runtime + broadcast channel) · `serde` / `serde_json` · `async-trait` · `thiserror` · `tracing` + `tracing-subscriber` · `uuid` v4 · `chrono` UTC · `axum` (websocket server) · `clap` (CLI args) · vanilla HTML / CSS / JS (no framework) for the PWA.

**Source spec:** [`docs/specs/2026-05-13-1bit-cashier-design.md`](../specs/2026-05-13-1bit-cashier-design.md). Every task in this plan implements a piece of that spec; deviations would invalidate the plan.

```rust
// 1bit.cashier — Cash-only autonomous POS
// Copyright (C) 2026 1bit.cashier contributors
// SPDX-License-Identifier: AGPL-3.0-only

use crate::bus::EventBus;
use crate::error::BusError;
use crate::events::EventEnvelope;
use async_trait::async_trait;
use tokio::sync::broadcast::{self, Receiver, Sender};

#[derive(Clone)]
pub struct MemoryBus {
    tx: Sender<EventEnvelope>,
}

impl MemoryBus {
    pub fn new(capacity: usize) -> Self {
        let (tx, _rx) = broadcast::channel(capacity);
        Self { tx }
    }
}

#[async_trait]
impl EventBus for MemoryBus {
    async fn publish(&self, env: EventEnvelope) -> Result<(), BusError> {
        match self.tx.send(env) {
            Ok(_) => Ok(()),
            // No active subscribers is not an error for v0.1 — the bench
            // path may publish before any consumer attaches. The bus
            // semantics: receivers only see envelopes published after they
            // subscribed. Document this in `EventBus::publish`'s doc later
            // if it becomes load-bearing.
            Err(_) => Ok(()),
        }
    }

    fn subscribe(&self) -> Receiver<EventEnvelope> {
        self.tx.subscribe()
    }
}
```

```rust
    pub fn change_qty(&mut self, sku: &str, qty: u32) -> Result<(), OrderError> {
        let idx = self
```

```rust
    pub fn apply(&mut self, t: Transition) -> Result<(), OrderError> {
        use OrderState::*;
        use Transition::*;

        let next = match (self.state, t) {
            (Idle, StartBuilding) => Building,
            (Building, StartReview) => Review,
            (Review, StartAwaitingPayment) => AwaitingPayment,
            (AwaitingPayment, StartPaying) => Paying,
            (Paying, Finalize) => Complete,
            (Complete, Reset) => Idle,
            (Idle, Reset) => Idle,
            (from, t) => {
                return Err(OrderError::InvalidTransition {
                    from: from.as_str().to_string(),
                    to: format!("{:?}", t),
                });
            }
        };

        self.state = next;
        Ok(())
    }
```

```rust
// Copyright (C) 2026 1bit.cashier contributors
// SPDX-License-Identifier: AGPL-3.0-only

use crate::cart::Cart;
use crate::state::{OrderState, StateMachine, Transition};
use cashier_bus::events::{Event, EventEnvelope, IntentAction, OrderLine, OrderUpdate};
use cashier_bus::language::Language;
use cashier_bus::EventBus;
use chrono::Utc;
use std::collections::HashMap;
use std::sync::Arc;
use tokio::task::JoinHandle;
use uuid::Uuid;

#[derive(Debug, Clone)]
pub struct CatalogEntry {
    pub sku: String,
    pub name: String,
    pub unit_price_cents: u64,
}

#[derive(Debug, Clone, Default)]
pub struct Catalog {
    by_sku: HashMap<String, CatalogEntry>,
}

impl Catalog {
    pub fn from_entries(entries: Vec<CatalogEntry>) -> Self {
        let mut by_sku = HashMap::new();
        for entry in entries {
            by_sku.insert(entry.sku.clone(), entry);
        }
        Self { by_sku }
    }

    pub fn lookup(&self, sku: &str) -> Option<&CatalogEntry> {
        self.by_sku.get(sku)
    }
}

pub struct OrderEngine {
    handle: JoinHandle<()>,
}

impl OrderEngine {
    pub fn spawn<B>(bus: Arc<B>, catalog: Catalog) -> Self
    where
        B: EventBus + 'static,
    {
        let mut rx = bus.subscribe();
        let bus_for_task = bus.clone();
        let handle = tokio::spawn(async move {
            let mut cart = Cart::new();
            let mut machine = StateMachine::new();
            let mut current_tx: Option<Uuid> = None;

            while let Ok(env) = rx.recv().await {
                let dirty = match &env.payload {
                    Event::VisionDetection(v) => {
                        if let Some(sku) = &v.sku {
                            if let Some(entry) = catalog.lookup(sku) {
                                if machine.state() == OrderState::Idle {
                                    let _ = machine.apply(Transition::StartBuilding);
                                }
                                current_tx.get_or_insert(env.transaction_id);
                                cart.add_item(&entry.sku, &entry.name, 1, entry.unit_price_cents);
                                true
                            } else {
                                tracing::warn!("vision: unknown sku {}", sku);
                                false
                            }
                        } else {
                            false
                        }
                    }
                    Event::AudioIntent(intent) => {
                        let mut changed = false;
                        for action in &intent.actions {
                            match action {
                                IntentAction::Add { sku, qty } => {
                                    if let Some(entry) = catalog.lookup(sku) {
                                        cart.add_item(&entry.sku, &entry.name, *qty, entry.unit_price_cents);
                                        changed = true;
                                    }
                                }
                                IntentAction::Remove { sku } => {
                                    if cart.remove_item(sku).is_ok() {
                                        changed = true;
                                    }
                                }
                                IntentAction::ChangeQty { sku, qty } => {
                                    if cart.change_qty(sku, *qty).is_ok() {
                                        changed = true;
                                    }
                                }
                                IntentAction::Void => {
                                    cart = Cart::new();
                                    let _ = machine.apply(Transition::Reset);
                                    current_tx = None;
                                    changed = true;
                                }
                                IntentAction::Clarify { question } => {
                                    tracing::info!("audio clarify: {}", question);
                                }
                            }
                        }
                        changed
                    }
                    _ => false,
                };

                if dirty {
                    let lines = cart
                        .lines()
                        .iter()
                        .map(|l| OrderLine {
                            sku: l.sku.clone(),
                            name: l.name.clone(),
                            qty: l.qty,
                            unit_price_cents: l.unit_price_cents,
                            line_total_cents: l.line_total_cents(),
                        })
                        .collect();
                    let update = OrderUpdate {
                        state: machine.state().as_str().to_string(),
                        lines,
                        total_cents: cart.total_cents(),
                    };
                    let envelope = EventEnvelope {
                        ts: Utc::now(),
                        transaction_id: current_tx.unwrap_or_else(Uuid::new_v4),
                        correlation_id: env.correlation_id,
                        language: env.language,
                        payload: Event::OrderUpdate(update),
                    };
                    let _ = bus_for_task.publish(envelope).await;
                }
            }
        });
        Self { handle }
    }

    pub fn join_handle(&self) -> &JoinHandle<()> {
        &self.handle
    }
}
```

```rust
// 1bit.cashier — Cash-only autonomous POS
// Copyright (C) 2026 1bit.cashier contributors
// SPDX-License-Identifier: AGPL-3.0-only

use axum::extract::ws::{Message, WebSocket, WebSocketUpgrade};
use axum::extract::State;
use axum::response::IntoResponse;
use axum::routing::get;
use axum::Router;
use cashier_bus::events::Event;
use cashier_bus::memory::MemoryBus;
use cashier_bus::EventBus;
use futures::SinkExt;
use std::net::SocketAddr;
use std::path::PathBuf;
use std::sync::Arc;
use tokio::net::TcpListener;
use tower_http::services::ServeDir;

#[derive(Clone)]
struct AppState {
    bus: Arc<MemoryBus>,
}

pub async fn serve(
    bus: Arc<MemoryBus>,
    port: u16,
    static_dir: PathBuf,
) -> anyhow::Result<()> {
    let state = AppState { bus };
    let app = Router::new()
        .route("/ws", get(ws_handler))
        .fallback_service(ServeDir::new(static_dir))
        .with_state(state);

    let addr = SocketAddr::from(([0, 0, 0, 0], port));
    let listener = TcpListener::bind(addr).await?;
    tracing::info!("display server listening on http://{addr}");
    axum::serve(listener, app).await?;
    Ok(())
}

async fn ws_handler(
    ws: WebSocketUpgrade,
    State(state): State<AppState>,
) -> impl IntoResponse {
    let bus = state.bus.clone();
    ws.on_upgrade(move |socket| handle_socket(socket, bus))
}

async fn handle_socket(socket: WebSocket, bus: Arc<MemoryBus>) {
    use futures::stream::StreamExt;
    let (mut sender, mut receiver) = socket.split();
    let mut rx = bus.subscribe();

    let forwarder = tokio::spawn(async move {
        while let Ok(env) = rx.recv().await {
            // Only forward OrderUpdate and CashEvent to the display
            let forward = matches!(env.payload, Event::OrderUpdate(_) | Event::CashEvent(_));
            if !forward {
                continue;
            }
            let json = match serde_json::to_string(&env) {
                Ok(s) => s,
                Err(e) => {
                    tracing::warn!("ws serialize error: {e}");
                    continue;
                }
            };
            if sender.send(Message::Text(json)).await.is_err() {
                break;
            }
        }
    });

    // Drain incoming messages (display does not currently send anything; we
    // read so the socket close events are observed).
    while let Some(msg) = receiver.next().await {
        if matches!(msg, Ok(Message::Close(_)) | Err(_)) {
            break;
        }
    }
    forwarder.abort();
}
```

```html
      document.getElementById("lang-toggle").textContent = t("language_toggle");

      const u = state.lastUpdate;
      if (!u) {
        document.getElementById("state").textContent = t("idle");
        document.getElementById("lines").innerHTML = "";
        document.getElementById("total-amount").textContent = fmtMoney(0);
        return;
      }
      const stateKey = u.state.replace("-", "_");
      document.getElementById("state").textContent = t(stateKey);
      document.getElementById("lines").innerHTML = u.lines
        .map((l) => `<tr class="line">
            <td>${l.name}</td>
            <td>${l.qty}</td>
            <td>${fmtMoney(l.unit_price_cents)}</td>
            <td>${fmtMoney(l.line_total_cents)}</td>
          </tr>`)
        .join("");
      document.getElementById("total-amount").textContent = fmtMoney(u.total_cents);
    }

    function showCashEvent(env) {
      const c = env.payload;
      const el = document.getElementById("cash-event");
      el.style.display = "block";
      el.textContent =
        `${fmtMoney(c.tendered_cents)} tendered  →  ${fmtMoney(c.change_cents)} change`;
    }

    function connect() {
      const proto = location.protocol === "https:" ? "wss" : "ws";
      const ws = new WebSocket(`${proto}://${location.host}/ws`);
      ws.onmessage = (msg) => {
        try {
          const env = JSON.parse(msg.data);
          if (env.payload?.kind === "order_update") {
            state.lastUpdate = env.payload;
            render();
          } else if (env.payload?.kind === "cash_event" && env.payload.cash === "charged") {
            showCashEvent(env);
          }
        } catch (e) {
          console.warn("bad message", e);
        }
      };
      ws.onclose = () => setTimeout(connect, 1000);
    }

    document.getElementById("lang-toggle").addEventListener("click", () => {
      state.locale = state.locale === "en" ? "mikq-LJ" : "en";
      localStorage.setItem(STORAGE_KEY, state.locale);
      render();
    });

    loadLocales().then(connect);
  </script>
</body>
</html>
```

```markdown
# Architecture

This document is the evergreen summary of how 1bit.cashier is put together at the current release. For the original design rationale and the deltas-from-handoff that produced this shape, see [`specs/2026-05-13-1bit-cashier-design.md`](specs/2026-05-13-1bit-cashier-design.md).

## At-a-glance

1bit.cashier is a Cargo workspace of six Rust library crates plus one binary plus a single-file PWA. Every subsystem talks to every other subsystem through a single `EventBus`. In v0.1.0-bench the bus is an in-process `tokio::sync::broadcast` channel; in a later release a NATS backend may be wired in behind a feature flag (spec §13.D1).

```
              ┌────────────┐
              │ EventBus   │ (tokio::broadcast in v0.1)
              └─────┬──────┘
       publishes /  │  \ subscribes
        ┌────────┐  │  ┌───────────┐  ┌──────────────┐
        │ vision │──┤──│ order     │──│ axum ws ──▶  │ customer-display (PWA)
        │ audio  │──┤  │  engine   │  └──────────────┘
        │ recycler──┤  │  + cart   │
        │ audit  │◀─┴──│  + state  │
        └────────┘     └───────────┘
```

## Crates

| Crate              | Responsibility                                              | v0.1 impl                       |
| ------------------ | ----------------------------------------------------------- | ------------------------------- |
| `cashier-bus`      | `EventBus` trait + `EventEnvelope` + 5 typed event variants | tokio broadcast (`MemoryBus`)   |
| `cashier-order`    | `Cart`, `StateMachine`, `OrderEngine` (subscribes + publishes) | full TDD; 15 tests           |
| `cashier-vision`   | `Vision` trait + `MockVision`                               | scripted detection mock         |
| `cashier-audio`    | `Audio` trait + `MockAudio`                                 | scripted intent mock            |
| `cashier-recycler` | `Recycler` trait + `MockRecycler` + `ChargeOutcome`         | computes change in-memory       |
| `cashier-audit`    | `Audit` trait + `MemoryAudit`                               | subscribes to bus, records `Vec` |

## Binary

`cashier-pos` (in `apps/cashier-pos/`) parses CLI args, loads the catalog (env `CASHIER_CATALOG_PATH` overrides), wires the bus to mocks, runs the bench demo, and serves the customer display over an axum websocket.

## PWA

`apps/cashier-display/` is a single `index.html` plus two JSON locale files (`en.json`, `mikq-LJ.json`). The websocket client subscribes to `/ws`, renders `OrderUpdate` and `CashEvent` envelopes, and exposes a manual English ↔ Mi'kmaq Listuguj toggle. Mi'kmaq strings that still contain the `[mikq-LJ:` stub marker silently fall back to English at render time.

## Event flow (bench mode)

```
MockVision publishes VisionDetection events
  → OrderEngine adds SKUs to Cart, emits OrderUpdate
    → axum ws forwards OrderUpdate to PWA
    → MemoryAudit records every envelope
MockAudio publishes AudioIntent events
  → OrderEngine applies ChangeQty, emits OrderUpdate
    → ...
MockRecycler.charge() publishes CashEvent::Charged
  → axum ws forwards to PWA
```

## What does NOT exist yet (deferred to later releases)

- Real Glory CI-10 driver (v0.2.0)
- Real camera + VLM (v0.2.0)
- Real microphone + whisper.cpp + intent LLM (v0.2.0)
- sqlite-backed audit log (v0.2.0)
- Cashier-tablet override flows (v0.2.0)
- NATS-backed event bus (v0.5+, only if/when multi-host distribution lands)
- Full Mi'kmaq Listuguj locale strings (gated on community review, ROADMAP §5)
```
